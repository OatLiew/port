Alarm Clock

This is my first Android App. The objective of this project is to create an Ice Cream Sandwich-compatible Android launcher application that, when started, displays the current local time in 12-hour format with AM/PM denotation. The displayed time is to be aligned to the center of the screen vertically and horizontally. The launched activity fills the screen with a Black (Hex 000000) background without a title bar and the time text is displayed using a White (Hex FFFFFF) 48px Iceland font.


In addition, the application implements an option menu or button on the main screen that allows the creation and storage of a single alarm that allows the User to select a time to have the alarm fire. When the set alarm time is hit, the application will change the color of the current time from the default White (Hex FFFFFF) to Red (Hex FF0000) for 5 seconds. After 5 seconds, the color of the current time will revert back to White (Hex FFFFFF).
